# bikelog-2

Current version="0.1.0"


## Ako hoćeš unjeti novi artikl:
- unesi podatke o artiklu u "templating/data/articles.json"
. napravi novu stranicu u "templating/pages/article-NAME.hbs" 
- unesi podatke o početnoj stranici u "templating/data/tours.json"
- run `grunt dist`

### Requirements For Frontend Development

* NodeJS (>= 0.12.0)
* Grunt

## Ako hoćeš generisati fotografije:
- copiraj fotografije u folder assets/img/original u rezoluciji >1920
- run `grunt pictures`
